## Frontend start
Устанавливаем node.js <br>
cd ./my-app  <br>
npm install <br>
npm run dev <br>
<br>Открываем страницы:<br>
http://localhost:3000/search <br>
http://localhost:3000/tender


## Backend start
Запустить бд <br>
Поменять адрес, юзера и пароль в src/main/resources/application.properties <br>
./gradlew bootRun <br>
