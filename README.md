## Frontend start
1. Устанавливаем node.js <br>
2. cd ./my-app  <br>
3. npm install <br>
4. npm run dev <br>
<br>Открываем страницы:<br>
http://localhost:3000/search <br>
http://localhost:3000/tender


## Backend start
1. Запустить бд 
2. Поменять адрес, юзера и пароль в src/main/resources/application.properties 
3. ./gradlew bootRun
